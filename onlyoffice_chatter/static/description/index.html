ONLYOFFICE Chatter Integration

Create OnlyOffice documents directly from the Chatter of any Odoo record.

Key Features

Create Documents

Create DOCX, XLSX, and PPTX files from any Chatter.

Native Attachments

Files are stored as standard ir.attachment — no extra apps needed.

Community Edition

Works with Odoo Community. Does not require Enterprise Documents.

How It Works

Step 1. Click the "New" button on any Chatter.

Step 2. Choose a format and enter a title.

Supported Formats

.docx

Word Document

.xlsx

Excel Spreadsheet

.pptx

PowerPoint Presentation

Requirements

  • onlyoffice_odoo module installed and configured
  • ONLYOFFICE Document Server running and accessible
  • Odoo 19.0 Community or Enterprise

Multilingual

Available in English, Spanish, French, Russian, Chinese, and Dutch (Belgium).
