How to Use ONLYOFFICE Chatter

Step-by-step guide to create documents directly from Odoo Chatter.

Step 1: Verify app

Open any record (customer, order, task, etc.) and scroll down to the Chatter section at the bottom.

Step 2: Press New

Click the "New" button next to the Chatter actions to create a document.

Step 3: Create a Document

Click the "New" button next to the Chatter actions. Select the document format (Document, Spreadsheet, or Presentation), enter a title, and click "Create".

Tips

  • The document is automatically attached to the record as a standard ir.attachment
  • You can edit the document directly in ONLYOFFICE and changes are saved automatically
  • The integration works with any Odoo record that has Chatter enabled
  • No need for Odoo Enterprise Documents — works with Community Edition
